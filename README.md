## 🎯 What You'll Learn

By the end of this module, you'll understand:

✅ What are variables and why we need them  
✅ How to declare variables using `let`, `const`, and `var`  
✅ JavaScript Data Types (Primitive & Non-Primitive)  
✅ The difference between `null` and `undefined`  
✅ How to use the `typeof` operator  
✅ Best practices for variable declaration  
✅ Block scope and functional scope in JavaScript

## 🚀 How to Use This Repository

### Step 1: Clone the Repository
```bash
git clone https://github.com/yourusername/js.git
cd js
```

### Step 2: Open the File
```bash
# Using VS Code
code 01_basics/variables.js

# Or open in any text editor
```

### Step 3: Run the Code
```bash
# Using Node.js
node 01_basics/variables.js

# Output will show in terminal
```


## 🎓 Next Steps

After mastering variables and data types, learn:

1. **Operators** - Arithmetic, Logical, Comparison operators
2. **Conditionals** - if/else statements, switch statements
3. **Loops** - for, while, forEach loops
4. **Functions** - Creating and using functions
5. **Arrays** - Methods like map(), filter(), reduce()
6. **Objects** - Object methods and properties
7. **Async JavaScript** - Promises, async/await
